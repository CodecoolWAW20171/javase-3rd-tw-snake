# Snake

A [Snake game](https://en.wikipedia.org/wiki/Snake_(video_game_genre)) written in Java using JavaFX as a GUI.

## Features

* Single and versus game mode
* Various power-ups
* Killer lasers
* Menu bar
* Restart
* Pause

## Requirements

* Java 10
